# 📊 Sales & Profit Dashboard – Global Superstore

## 📌 Project Overview

This project presents an interactive dashboard analyzing sales and profit data from the Global Superstore dataset. It provides valuable insights into business performance across regions, categories, and time.

---

## 🚀 Features

* 💰 Total Sales, Profit, Quantity, Orders overview
* 🌍 Region-wise sales analysis
* 📦 Category-wise performance (Furniture, Office Supplies, Technology)
* 📈 Profit trends over time
* 🛍️ Product-level sales analysis
* 📊 Sales vs Profit comparison

1. Download this repository
2. Open the `.pbix` file in Power BI
3. Interact with the dashboard

---

## 📌 Key Insights

* Technology category generates highest profit
* Central region has the highest sales
* Profit trends show fluctuations over time
* Some categories have high sales but lower profit margins
